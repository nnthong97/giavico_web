# planning/common/

Shared code dùng chung cho tất cả sub-features (GV, AV, ND).

- Models & interfaces chung: xem `../models/planning.model.ts`
- Service chung: xem `../data-access/planning.service.ts`
- Utils chung (date helpers, formatters): thêm vào đây khi cần

Hiện tại common models/service vẫn giữ ở thư mục gốc để tránh breaking changes.
